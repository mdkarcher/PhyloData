# PhyloData
Repository of data files for phylodynamics research

## Datasets

1. **New York influenza** BEAST XML for inferring genealogy using sequence data from [1].
    * NewYork.xml

2. **Regional influenza** BEAST XML for inferring genealogy using sequence data from [2].
    * Europe.xml
    * India.xml
    * JapanKorea.xml
    * NorthChina.xml
    * Oceania.xml
    * SouthAmerica.xml
    * SouthChina.xml
    * SoutheastAsia.xml
    * USACanada.xml
